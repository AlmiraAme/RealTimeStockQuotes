 91      * Dit is de createRandom functie.
 92      * De variable oldStocks, roundToDouble, maxMinPrice, Percent, Newstocks moeten worden meegeven om de random stocks te maken.
 93      * Als eerst wordt er een loop gemaakt voor elke stock in de oude stock array ( om door elke stock heen te gaan)
 94      * De stock wordt er in een locale variable opgeslagen.
 95      * Eerst wordt er een nieuwe priceStock gemaakt door de functie getRandomPrice op te roepen en de return waarde op te slaan.
 96      * Dan wordt de verandering berekend, om ervoor de zorgen dat het een double wordt heb ik gebruik gemaakt van de website's hieronder.
 97      * De rekensom ja die ken ik nog van de middelbare school (wiskunde).
 98      * De verandering in procenten heb ik ook gebruik gemaakt van de math.round, De berekening ken ik van de middelbare school(economie)
 99      * Na elke stock wordt er dus de prijs verandert en de change + change in procent berekend en opgeslagen in de newStocks.
100      * Voor elke nieuwe stock wordt er ook een nieuwe StockModel gemaakt en de variables meegegeven.

103      * @param oldStocks : De array met oude stocks.
104      * @param roundToDouble : Het aantal nummers dat achter de komma mag.
105      * @param maxMinPrice : maximale/minimale price verandering.
106      * @param percent : Het aantal procent.
107      * @param newStocks : De array met nieuwe Stocks.
108      */
109     function createRandom(oldStocks, roundToDouble, maxMinPrice, percent, newStocks) {
110         var priceIndex = 1;
111         var symbolIndex = 0;
112         for (var i = 0; i < oldStocks.length; i++) {
113             var stock = oldStocks[i].stock;
114             var newPriceStock = getRandomPrice.call(this, roundToDouble, stock[priceIndex], maxMinPrice);
115             var newChange = Math.round(roundToDouble * (newPriceStock - stock[priceIndex])) / roundToDouble;
116             var newChangePercent = Math.round(roundToDouble * (((newPriceStock - stock[priceIndex]) / stock[priceIndex]) * percent)) / roundToDouble + '%';
117             newStocks[i] = new StockModel(stock[symbolIndex], newPriceStock, newChange, newChangePercent);
118         }
119     }
