Class Detail
StockModel(name, price, change, changePrecentage)
Dit is de constructor van de StockModel, De StockModel kan zo vaak als je wil worden aangemaakt. Ik maak er een constructor van om een specifieke object van te maken die zijn eigen methodes en variabels heeft. Wanneer een nieuwe stockmodel wordt gemaat moet de naam, prijs, verandering en procenten worden meegegven, zodat er een nieuwe stock array kan worden aangemaakt. De reden dat ik alle functies en variables in de constructor zet is om het zoveel mogenlijk op een class te laten lijken met zijn eigen methodes/variables. Om het erin te zetten zien het er ook meer uit als een class.

: Naam/symbool van een bedrijf.

: De verandering van een bedrijf.
